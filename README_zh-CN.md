## 安装

安装 vitepress-theme-async 等插件

```bash
npm install --save vitepress-theme-async
```

或者

```bash
yarn add --save vitepress-theme-async
```

## 更新

你只需要执行:

```bash
npm install vitepress-theme-async@latest
```
